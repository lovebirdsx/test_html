下面演示的是 CSS 选择器的优先级

!important的优先级最高

如果没有使用 !important，则 class 的顺序决定了优先级


      .danger {
        color: red;
      }
      .danger2 {
        color: red !important;
      }
      .base {
        background-color: lightgray;
        color: turquoise;
      }
